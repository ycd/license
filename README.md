### Compilation

license is written in [Rust](https://www.rust-lang.org).
You will need rust version [1.45.0](https://blog.rust-lang.org/2020/07/16/Rust-1.45.0.html) or higher.

To build, download the source code and run:

    $ cargo check
    $ cargo install --path .
